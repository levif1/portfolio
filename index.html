Fogle Peripherals

A single page ecommerce store made with react, commerce.js, stripe, material-ui. The site allows a store owner to use commerce js to manage the content displayed on the page. It uses stripe for the handling payment processing, and for storing customer addresses. Commerce.js is used for the backend storing of products, and the categories. material-ui is the front end formatting, similar to bootstrap.

Esports Dashboard

A google chrome extention that focuses on League of Legends esports. The dashboard has the schedule, and stats for all League pro games in North America (LCS), and Europe(LEC). It allows users to also add favorite links, so they can use a button to go to that website. All the data is grabbed from Leagues creator, Riot, via an API. The extention is created exclusively react.
